## 一、三个业态定义

### 奥莱（Outlet）

品牌货、打折卖。过季款、尾货、或奥莱专供款。

```
核心逻辑：同一品牌，正价 2000，奥莱 800。
消费者心态：「用更少的钱买到牌子」——性价比+品牌感。
```

### 传统百货

品牌各自租柜台，联营扣点。场地方不碰货，只收租+抽流水。

```
核心逻辑：品牌自己铺货、管库存、派导购。
消费者心态：「买件大衣就走」——目的明确，买完即走。
```

### 冒店（Mall / 购物中心）

口语中对 Mall 的方言化叫法。室内封闭式，餐饮+零售+影院。

```
核心逻辑：一站式消费目的地。
问题：同质化严重，每个 Mall 一样品牌、一样楼层布局。
```

## 二、数据对比（2025）

| 维度 | 传统百货 | Mall（冒店） | 奥莱 |
|------|---------|-------------|------|
| 客流方向 | ↘ 下降 | → 持平/微降 | ↗ 上升（+15.6%） |
| 坪效 | 低 | 中 | 高（宁波杉井 446 元/㎡/月，超部分万象城） |
| 消费者心态 | 我要买东西 | 我要逛一下 | 我要逛+买+发小红书 |
| 品牌依赖 | 强 | 强 | 中（消费者认性价比） |
| 增长曲线 | 负 | 平 | 陡 |
| 库存风险 | 品牌扛 | 品牌扛 | 低（尾货+专供款） |
| 资本认可 | 低 | 中 | REITs 抢着上 |

**结论：奥莱是当前最强业态。**

## 三、奥莱三大增长引擎

### 引擎一：仓储式奥莱下沉

- 京东奥莱：2024 年底 3 家 → 2025 年底 60 家 → 2026 年起年开 100+
- 荟品仓：全国 70+ 店，1-3 折买断制
- 打法：三到五线城市，轻资产，贴着社区走

### 引擎二：运动户外品类爆炸

- 奥莱运动品牌门店 4800+ 家，占零售业态 18.8%
- 宁波杉井：运动户外贡献 37.73% 营收
- 始祖鸟、lululemon、萨洛蒙、迪桑特——"中产四宝"引流

### 引擎三：第三代奥莱「微度假」

- 从折扣卖场 → 购物+社交+周末目的地
- 湾里·王府井 WellTown：首月 3 亿销售额、330 万客流
- Z 世代主力，80% 从抖音/小红书「种草→到店→发帖」

## 五、预算战略含义

老板的判断逻辑：奥莱最强，不是因为最大（传统百货总盘子可能更大），而是因为**在增长、效率、资本回报三维上同时领先**。

```
预算方向：
  奥莱 → 加码
  Mall → 守盈利店、关拖后腿的
  传统百货 → 不再加码
```

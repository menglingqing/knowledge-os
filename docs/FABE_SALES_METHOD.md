# FABE 销售法则

## 概述

FABE 是一套经典的产品介绍和销售说服框架，在服装行业中广泛用于导购培训、商品说明和营销内容设计。

## 四个层次

| 字母 | 含义 | 服装行业示例 |
|------|------|-------------|
| **F — Feature**（特征） | 产品的客观属性、参数、构成 | "这件西装是**100%美丽诺羊毛**，采用**半衬里工艺**" |
| **A — Advantage**（优势） | 这个特征比竞品/普通产品好在哪 | "比普通羊毛**更细软**，半衬里让它比全衬里西装**轻了40%**" |
| **B — Benefit**（利益） | 对顾客有什么**实际好处** | "夏天穿不闷热、不扎皮肤，出差折叠放箱子**不容易起死褶**" |
| **E — Evidence**（证据） | 用什么证明你说的 | "您摸摸这个面料（触觉体验），这是上周**爆款卖了300+件**的返单" |

## 核心逻辑

```
Feature → Advantage → Benefit → Evidence
   ↓           ↓           ↓           ↓
 "是什么"  → "为什么好" → "跟我有什么关系" → "凭什么信你"
```

**关键洞察**：大多数销售和产品描述都卡在 F/A 层（罗列参数、堆砌优势），没有拉到 B/E 层。FABE 的核心是把话术推到 **"这对你意味着什么"** 和 **"你可以亲眼确认"** 这两个层级。

## 服装行业特有的误区

1. **把 Feature 当 Benefit 讲** — "这是300支高支棉" 对顾客没意义，要翻译成"摸起来像丝绸，越洗越软"
2. **Evidence 太弱** — 不要只说"品质很好"，要用触觉、数据（复购率、退换率）、试穿对比来证明
3. **不区分客群** — 同一个 Feature 对不同人利益不同：年轻人更关心"显瘦/好搭"，商务客群更关心"抗皱/耐穿"

## 应用场景

- 导购话术设计
- 商品详情页文案
- 企划案的产品说明
- 直播带货脚本
- 商品培训材料

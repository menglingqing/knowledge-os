# AI 公司为什么用不了 PE

---

## 一、PE 是什么

```
PE（市盈率）= 股价 / 每股收益 = 市值 / 净利润

含义：市场愿意为你每赚 1 块钱利润，付多少钱。
PE = 20 → 回本需要 20 年。
```

---

## 二、为什么 AI 公司用不了

**分母是负数或接近于零，公式失效。**

```
传统 SaaS 公司：
  收入 $1B - 成本 $700M = 利润 $300M ✅ 有意义

AI 公司：
  收入 $1B
  - 算力成本 $2B（GPU、电费）
  - 研发成本 $1.5B（研究员年薪 $1M+）
  - 销售成本 $500M
  利润 = -$3B ❌ PE = 负数，无意义
```

---

## 三、三层深层原因

### 1. 算力是重资产，但收入不匹配

每回答一个问题，成本不是电费——是 GPU 跑了多少 token。按什么收钱？月订阅？按 token？定价模型没跑通，收入覆盖不了推理成本，更别提训练成本。全行业问题。

### 2. 研发投入前置，收入后置

训练一个前沿模型花几亿美金。建好了，每个用户问一次还要继续烧推理成本。传统软件边际成本趋近于零，AI 边际成本是正的——多一个用户、多一次调用、多一笔 GPU 账单。

### 3. 竞争把定价压在地板上

DeepSeek 基本免费、OpenAI 不停降价、Google 把 Gemini 塞进免费 Workspace。一家降价，全行业利润率归零。利润率无法收敛到正数，PE 没有任何预测能力。

---

## 四、投资者用什么替代 PE

| 指标 | 公式 | 为什么用 |
|------|------|---------|
| **EV/Revenue** | 企业价值 / 收入 | 不要求盈利，只看规模 |
| **EV/ARR** | 企业价值 / ARR | SaaS + AI 混合指标 |
| **Rule of 40** | 收入增速% + 利润率% | 增长和效率的平衡 |
| **Gross Margin** | (收入 - 直接成本) / 收入 | 最恐怖指标——推理成本吃掉毛利 |
| **DCF** | 未来现金流折现 | 赌终局时利润会来 |

```
传统：你现在赚多少？（PE 回答）
AI：  你活到终局时能赚多少？（DCF 猜）
      你现在规模多大？（EV/Revenue 回答）
      你增速多快？（Rule of 40 回答）
      你单位经济能不能收敛到正数？（Gross Margin 回答）
```

---

## 五、ARR 作为光的隐喻

ARR 这个指标本身是一束光。它照亮了「续费」维度，但把「毛利能不能转正」「用户为什么续」「用户每个月到底用了多少次推理」这些全推到了暗处。

AI 公司的真实处境在场域层比 ARR 层复杂得多，只看 ARR 做投资决策和 PE 一样不够——需要转不同方向的光去照亮不同角落。

**ARR 是光，不是场域。**
